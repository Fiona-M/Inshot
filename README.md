# 迭代历史
### v1.162     2014-12-11
- 名字叫InstaShot
- 首页：视频编辑、图片入口、系统录像入口、系统拍照入口
    - 视频编辑：
        - 剪切、位置、背景、旋转、翻转
        - 只能选择单个视频
        - 保存时选择分辨率
        - 分享到Instagram快捷入口
    - 图片编辑：
        - 位置、背景、旋转、翻转
        - 只能选择单个图片
