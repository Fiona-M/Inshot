# 迭代历史
### v1.162     2014-12-11
- 名字叫InstaShot
- 首页：视频编辑、图片入口、系统录像入口、系统拍照入口
    - 视频编辑：
        - 剪切、位置、背景、旋转、翻转
        - 只能选择单个视频
        - 保存时选择分辨率
        - 分享到Instagram快捷入口
    - 图片编辑：
        - 位置、背景、旋转、翻转
        - 只能选择单个图片

### v1.233     2015-1-29
- 视频编辑：
    - 新增Zoom in/out
    - 新增音乐（只有一首）
- 图片编辑：
    - 新增模糊背景，分三个层次Light Medium Heavy
    - 新增滤镜
- 支持语言选择
- 支持Video Codec选择：H.264或MPEG-4
- 保存路径显示和支持跳转

### v1.251     2015-3-19
